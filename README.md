# Мониторинг доступности сайтов 
Программа для отслеживания доступности веб-ресурсов с визуализацией и опциональным логированием. 

---

## 🚀 Запуск программы

### Для тех, кто не хочет запариваться:
1. Скачайте файл `monitor.exe` из раздела [Releases](https://github.com/ServChan/WebsiteAccessibilityMonitor/releases).
2. Запустите его. При появлении предупреждения выберите "Подробнее" → "Всё равно выполнить".

### Для любителей запускать все руками:
1. Убедитесь, что установлен Python 3.7+.
2. Установите зависимости:
   ```bash
   pip install aiohttp rich colorama
Запустите скрипт:

```bash
python access.py
```

⚙️ Настройка
Файл config.json создаётся автоматически. Пример конфигурации:
```json
{
  "monitor_settings": {
    "interval": 60,
    "timeout": 5,
    "valid_status_codes": [200, 301, 302],
    "sorted": "status"
  },
  "websites": ["google.com", "github.com"],
  "Monitor": {
    "logging_enabled": true,
    "log_file_path": "logs/monitor.log"
  }
}
```
